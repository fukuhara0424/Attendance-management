# 勤怠管理システム

スタッフの出退勤打刻・休暇申請・管理者による勤怠確認ができるWebアプリです。

## 機能

- **打刻ポータル**：出勤・休憩・退勤の記録（PIN認証）
- **休暇申請**：年次有給・病欠・誕生日休暇・振替休暇・学校行事
- **管理者画面**（パスワード保護）
  - 従業員管理・家族情報
  - 出退勤状況・打刻履歴
  - 休暇申請の承認/却下
  - 休暇カレンダー（月別表示）
  - 誕生日・学校行事通知

## 使い方

### 打刻する（スタッフ）
1. トップ画面で「打刻する」を選択
2. 自分の名前を選んで「次へ」
3. 4桁のパスコードを入力
4. 出勤・休憩・退勤ボタンを押す

### 管理画面
1. トップ画面で「管理画面」を選択
2. 管理者パスワードを入力（初期値: `admin1234`）
3. 各タブから操作

> ⚠️ 初回ログイン後、設定タブからパスワードを変更してください。

## データの保存について

データはブラウザの `localStorage` に保存されます。
**同じブラウザ・同じ端末**で使い続ける限りデータは保持されます。

複数端末で共有したい場合は、別途バックエンド（Supabase等）との連携が必要です。

## GitHub Pages への公開手順

```bash
git clone https://github.com/あなたのユーザー名/kintai.git
cd kintai
# index.html を配置済みの状態で
git add .
git commit -m "initial commit"
git push origin main
```

その後 GitHubリポジトリ → Settings → Pages → Source を `main` ブランチの `/(root)` に設定。

公開URL: `https://あなたのユーザー名.github.io/kintai/`
